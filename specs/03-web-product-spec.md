# Web Product Spec

## 文档状态

- Status: Active
- Scope: Web 单页应用的用户功能与界面职责

## 产品目标

Web 是当前主交付物，必须满足：

- 可独立静态部署
- 默认本地离线可用
- 支持快速导入与验证码复制
- 可选接入 WebDAV 做远端同步

## 页面职责

### 主页面

负责：

- 展示所有条目
- 展示倒计时与验证码有效窗口
- 新增、编辑、删除条目
- 多选并批量分享

### 导入入口

负责：

- 文本导入 `otpauth://...`
- 摄像头扫码导入
- 图片二维码导入
- 处理 `?import=` 分享链接

### 分享对话框

负责：

- 展示原始 `otpauth://...` 列表
- 复制原始内容
- 生成当前域名对应的导入链接
- 预留加密分享入口

### WebDAV 卡片

负责：

- 输入 `WebDAV Host / Account / Password / Vault Secret`
- 验证配置
- 拉取远端快照覆盖本地
- 推送当前本地快照
- 刷新远端状态
- 清空配置

## 体验要求

### 验证码

- 默认显示连续数字，不插空格
- 支持点击复制
- 显示剩余有效秒数
- 自动刷新节奏稳定

### 导入

- 无效密钥必须有明确错误提示
- 导入结果要区分成功、跳过、失败
- 重复导入时避免无意义重复写入

### 同步

- WebDAV 配置要能本地记住
- `Vault Secret` 缺失时不允许拉取/推送
- 拉取前如果本地已有数据，必须提示覆盖风险
- revision 冲突必须有明确提示
